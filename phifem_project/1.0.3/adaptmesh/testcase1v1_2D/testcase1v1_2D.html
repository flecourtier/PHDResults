Cas test 1 : Carré + Poisson + Dirichlet (basse fréquence)

On considère le problème de Poisson avec condition de Dirichlet défini par

\[\left\{ \begin{aligned} -\Delta u &= f, \; &&\text{dans } \; \Omega, \\ u&=g, \; &&\text{sur } \; \Gamma. \end{aligned} \right.\]

On définit \(\Omega\) comme étant le carré défini par

\[\Omega=[-0.5 \pi, 0.5 \pi]^2\]

On considèrera pour ce cas test la solution analytique \(u_{ex}\) défini par

\[u_{ex}(x,y)=\exp(-((x-\mu_1)^2+(y-\mu_2)^2)/2)\times\sin(2x)\times\sin(2y)\]

homogène sur \(\Omega\) (\(g=0\)) et \(\mu_1,\mu_2\sim\mathcal{U}(-0.5, 0.5)\).
